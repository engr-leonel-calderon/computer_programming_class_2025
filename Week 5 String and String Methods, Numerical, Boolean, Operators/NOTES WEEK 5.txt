str(anyValue) - convert other datatype to string value

len(strValue) - return int length of strValue (how may chars)

strValue[index] - return char based on given index(int)

strValue.upper() - convert each letter to upper case

strValue.lower() - convert each letter to lower case

strValue.capitalize() - convert first letter of each word to upper case

strValue.split("subStr") - split string based on given char or substring return list

strValue.replace("subStr1", "subStr2") - replace substr1 by substr2

strValue.count("subStr") - count the char or substr present on string

strValue.index("char") - return the lowest index where char  is found

strValue.index("char", start, end) - return the lowest index where char  is found within range

strValue[start:end] - return substring or partial part of string based on include:exclude chars

strValue[start:end:step] - return substring or partial part of string based on include:exclude chars with steps
